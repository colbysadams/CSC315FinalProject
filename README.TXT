CSC 315 FINAL PROJECT BUILD


To run the project from the command line, go to the dist folder and
type the following:


java -jar "CSC315FinalProject.jar" tables.txt data.txt queries.txt


the tables.txt, data.txt, and queries.txt files are all included in the dist
folder.  If you want to run files from a different folder than the .jar file
you must include the full class path to those files.


the actual .java file is located at: /src/csc315finalproject/CSC315FinalProject.java


The write up is the word document: FinalProjectWriteUp.doc


The EER diagram is the workbench file: CSC315FinalProjectEER.mwb
